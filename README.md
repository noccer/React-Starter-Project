# Simple React Starter Project

> It does what it says on the tin.

## Instructions

Runs off node. So install that first.

Then install dependencies:

```
npm install
```

(Installs these dependencies)

- babel-core
- babel-loader
- babel-preset-react
- html-webpack-plugin
- webpack
- webpack-dev-server

Then build the project:

```
npm run build
```

Then run a server:

```
npm run start
```

Then go build some React stuffs.
